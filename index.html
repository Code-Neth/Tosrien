រៀនអំពី HTML

សូម្បីតែអ្នកចាប់ផ្ដើមចង់រៀន HTML ដោយចាប់ពីកម្រិតដំបូងទៅកម្រិតខ្ពស់ (0-100) ខាងក្រោមនេះជាគន្លឹះនិងវិធីសាស្រ្តដែលអ្នកអាចអនុវត្តបាន៖

1. កម្រិតដំបូងនៃ HTML (0-25)
>ដឹងថា HTML គឺជាអ្វី
>HTML ជាភាសាសម្គាល់ (HyperText Markup Language) ដែលប្រើសម្រាប់បង្កើតទំព័រប្រអប់បណ្តាញ។
រៀបចំបរិស្ថានការងារ
>ដំឡើងកម្មវិធីដូចជា Visual Studio Code, Notepad++, ឬអាចប្រើកន្លែងកូដអនឡាញដូចជា CodePen ឬ JSFiddle។
>រៀនរចនាសម្ព័ន្ធ HTML មូលដ្ឋាន
នេះគឺជាឧទាហរណ៍សម្ព័ន្ធ HTML មូលដ្ឋាន៖


ប្រធានបទចម្បង:
.Tags: h1, p, a, img, ។
. Attributes: href, src, alt ។

2. HTML កម្រិតមធ្យម (25-50)

> សិក្សាពី Tags ដ៏សំខាន់ៗ
. យល់ដឹងអំពី Semantic Tags ដូចជា header, footer, section, article និង aside។
. ការប្រើប្រាស់ Forms រៀនអំពី form, input, textarea, button និង attributes ដូចជា name, method, និង action។


> សរសេរដំណើរការមេឌា(Media Elements):
.​​ រូបភាពដោយ <img>
. វិដេអូដោយ video
. អូឌីយ៉ូដោយ audio
>ការចំណងជើង និង Navigation
បង្កើតតារាងដំណើរការជាមួយ a និង nav។

3. HTML កម្រិតខ្ពស់ (50-75)

> តារាង (Tables)
រៀនបង្កើតតារាងទិន្នន័យដោយ table, tr, td, th


> សរសេរដំណើរការមេឌា(Media Elements):
.​​ រូបភាពដោយ img
. វិដេអូដោយ video
. អូឌីយ៉ូដោយ audio
>ការចំណងជើង និង Navigation
បង្កើតតារាងដំណើរការជាមួយ a និង nav។
> HTML5 API
រៀនអំពីអត្ថកម្មដូចជា:
. Geolocation API
. Drawing Canvas
. Local Storage
Accessibility (ការចូលប្រើ)
ប្រើ Attribute aria-* និង Semantic Tags ដើម្បីធ្វើឱ្យវាសមស្របសម្រាប់អ្នកទាំងអស់។

រៀនអំពី CSS

សូម្បីតែអ្នកចាប់ផ្ដើមចង់រៀន CSS ដោយចាប់ពីកម្រិតដំបូងទៅកម្រិតខ្ពស់ (0-100) ខាងក្រោមនេះជាគន្លឹះនិងវិធីសាស្រ្តដែលអ្នកអាចអនុវត្តបាន៖

1. រៀនស្តាប់មូលដ្ឋានល្អ
>ដឹងថា CSS គឺជាអ្វី
>CSS (Cascading Style Sheets) គឺជាភាសាមួយដែលប្រើសម្រាប់តុបតែង និងរចនាទម្រង់គេហទំព័រ។ វាដំណើរការរួមគ្នាជាមួយ HTML (Hypertext Markup Language) ដើម្បីធ្វើអោយគេហទំព័រមានភាពស្រស់ស្អាត និងងាយស្រួលប្រើប្រាស់។

2. CSS អាចធ្វើអ្វីបានខ្លះ?
>តុបតែងមាតិកា (Style Content) CSS គ្រប់គ្រងរបៀបដែលធាតុ HTML មើលឃើញ ដូចជាការផ្លាស់ប្តូរពណ៌ អក្សរ ទំហំ និងចន្លោះ។ ឧទាហរណ៍៖
រៀបចំ Layout សម្រាប់គេហទំព័រ

>CSS ជួយរៀបចំទម្រង់នៃគេហទំព័រ ដូចជា Header, Footer, Sidebar និងមាតិកាចម្បង ដោយប្រើបច្ចេកទេស Flexbox និង Grid។
> ធ្វើឱ្យគេហទំព័រឆ្លើយតបទៅនឹងឧបករណ៍ផ្សេងៗ
> CSS អាចធ្វើអោយគេហទំព័រត្រូវនឹងទំហំអេក្រង់ផ្សេងៗ (Mobile, Tablet, Desktop)។


2.CSS ដំណើរការបែបណា?
.អ្នកអាចបញ្ចូល CSS ទៅក្នុងគេហទំព័របាន ៣ វិធី៖
. Inline CSS (ដាក់ CSS នៅក្នុង Tag HTML)
.Internal CSS (ដាក់ CSS នៅក្នុង Style tag នៅក្នុងឯកសារ HTML)

.External CSS (ដាក់ CSS នៅក្នុងឯកសារផ្សេង styles.css)
